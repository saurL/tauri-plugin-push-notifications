## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-get-fcm-token`
- `allow-get-apns-token`
- `allow-registerListener`
- `allow-unregister-listener`
- `allow-checkPermissions`
- `allow-requestPermissions`
